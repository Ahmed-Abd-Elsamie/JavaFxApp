# Maintenance Management

## Desktop App to Manage Computer Maintenance Process


### Features
</br>
1 - Print Bill
</br>
2 - Elastic Search
</br>
3 - Edit / Add Bill
</br>
4 - Add Hardware Device
</br>
5 - Add Software Device
</br>
6 - Add Printer Device
</br>

### Technologies

</br>
2 - FXML
</br>
3 - MySql Database
</br>
4 - PHP
</br>
